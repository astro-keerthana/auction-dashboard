# AuctionPro Live — Phase 2
### Professional Live Player Auction Management System

## Quick Start

### 1. Install Node.js
Download from https://nodejs.org/ (v18 or higher)

### 2. Install & Run
```bash
cd auctionpro-phase2
npm install
npm start
```
Open http://localhost:3000

**Default login: admin / admin123**

## How to Use

**Step 1** — Log in → Go to Events → Create event → Click Select

**Step 2** — Event Setup → Import players/teams CSV or add manually

**Step 3** — Auction Control → Start Auction → Control bids → Mark SOLD/UNSOLD

**Step 4** — Live Display tab → Open on projector in full screen (F11)

**Step 5** — Export → Download results CSV after auction

## CSV Format

**Players:** Name, Role, Base Price (L), Auction Order, Photo URL

**Teams:** Name, Short Name, Purse (L), Max Players, Color (#hex), Logo URL

## Multi-Device
- Admin: http://localhost:3000
- Projector/Mobile: http://[YOUR-IP]:3000 (same WiFi)
- Find IP: ipconfig (Windows) or ifconfig (Mac/Linux)

## Data
All data saved in data/auctionpro.db — persists through restarts.

Default: admin / admin123
